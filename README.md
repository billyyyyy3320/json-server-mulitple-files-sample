# json-server-mulitple-files-sample


## Build
`yarn`

Install dependencies

`yarn mock`

Start the server


## Easily mock API:

Add files:
```
└── mock
    └── hello
        ├── world.js
        └── index.js
```
And you'll be able to get data by visiting:

`localhost:9898/hello/world`
`localhost:9898/hello`
